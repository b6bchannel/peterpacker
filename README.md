## Peter Pack-er: Travel Packing List

一个为旅行前打包而做的轻量清单工具。

它不是复杂的行程管理、库存系统或购物清单，而是帮助你回答三个最实际的问题：

- 这次旅行要带什么？
- 每样东西装在哪个包里？
- 出发前还有什么没收拾？
- 旅行的每一天该穿什么？

## 你可以用它做什么

### 创建旅行清单

根据旅行范围、季节、住宿晚数、电子设备、可选模块和本次携带箱包，快速生成一份打包清单。

清单会自动包含常见基础物品，也可以按需要加入自驾、商务、防晒、化妆、长途飞行等模块。

### 像待办事项一样打包

在执行页中，每件物品都是一条彩色清单项：

- 右滑：标记为已打包
- 左滑：标记为本次不带
- 点击名称：修改物品名
- 点击数量：修改数量
- 点击箱包图标：切换收纳位置

已打包和本次不带的物品会折叠到底部，让主清单越来越短。

### 按类目或箱包查看

默认按物品类目查看，例如证件、数码、洗护、衣物、飞行出行等。

也可以切换到箱包视图，按 28 寸行李箱、双肩包、随身小包等收纳位置核对，适合出发前最后检查。

### 管理自己的物品库

应用内有一个轻量的“我的库”：

- 模块库：管理自驾、商务、防晒等可选模块
- 物品库：管理默认物品、分类和推荐收纳位置
- 箱包库：管理箱包名称、图标和属性

手动添加的物品可以只用于本次旅行，也可以保存进物品库。

### 每日穿搭

如果开启“安排穿搭”，可以为当前旅行安排：

- 出发当天穿什么
- 第 2 天、第 3 天等每天穿什么
- 上装、下装、鞋与配饰

穿搭只服务于当前旅行，不会变成复杂衣橱系统。

### 规则与备份

可以调整常见数量规则，例如：

- T 恤每几晚带一件，最多几件
- 内裤、袜子、洗衣片的数量规则
- 长裤、短下装、长袖、睡衣的默认数量规则

数据保存在当前浏览器本地，也可以导出 JSON 备份，之后再导入恢复。

## 数据与隐私

这个应用的数据默认保存在你的浏览器本地。

不会上传旅行清单、物品库、箱包或穿搭安排到服务器。

如果更换设备、清理浏览器数据或换浏览器使用，请先导出备份。

## 适合谁

适合经常旅行、出差、短途周末游、长途旅行，或者希望出门前减少遗漏的人。

它更像一个个人打包助手：轻量、直接、可反复使用。
